# Database vettoriale per ricerche di similarity

Vorrei creare una seplice applicazione in python utilizzando langchain

- Consulta sul sito langchain le informazioni sull'ultima release del framework
- Configura il progetto python con uv (non pip)
- Crea un semplice vectorstore persistente sul file system locale
- Sviluppa una utility per vettorializzare tutti i documenti in formato .md che si trovano in una directory (da creare) e popolare ilo vectorstore
- Sviluppa una utility cli per testare la ricerca per similarity dal vectorstore data una query
