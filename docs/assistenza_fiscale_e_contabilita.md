# Servizi alle Imprese – Assistenza Fiscale e Contabilità

**Parte del file genitore**: [servizi_alle_imprese.md](servizi_alle_imprese.md)

Servizio completo di consulenza **fiscale, contabile e tributaria**, erogato tramite **CNA CAF Lazio**, intermediario abilitato per gli invii telematici.

## Promozioni
- **30% di sconto** sul primo anno (associati CNA)  
- **40% di sconto** sul primo anno (chi utilizza già altri servizi CNA)

# 1. Servizi di Contabilità

## 1.1 Contabilità Ordinaria
**Destinatari**: società di capitali, società di persone, ditte individuali sopra soglia.  
**Servizi**:
- Registri IVA (acquisti/vendite)  
- Libro giornale  
- Libro inventari  
- Registro beni ammortizzabili  
- Scritture contabili  
- Situazioni periodiche patrimoniali ed economiche  

## 1.2 Contabilità Semplificata
**Destinatari**: ditte individuali e professionisti.  
**Servizi**:
- Registri IVA  
- Registro incassi/pagamenti  
- Monitoraggio entrate/uscite  

# 2. Adempimenti IVA
- Liquidazioni mensili o trimestrali  
- Compilazione LIPE  
- Versamenti F24  
- Monitoraggio scadenze  

# 3. Bilanci
**Bilancio IV Direttiva CEE**:
- Stato Patrimoniale  
- Conto Economico  
- Nota Integrativa  
- Allegati e libro inventari  

**Tipologie gestite**: bilanci consuntivi, previsionali, periodici.

# 4. Dichiarazioni Fiscali

## 4.1 Redditi
Modelli: **SC, SP, PF**, 730 tramite CAF.  
Servizi: compilazione, verifica, invio telematico.

## 4.2 IVA Annuale
- Compilazione dichiarazione  
- Calcolo IVA a debito/credito  
- Invio telematico  

## 4.3 IRAP
Calcolo base imponibile, modello completo.

## 4.4 ISA
- Compilazione indici  
- Verifica congruità  
- Ottimizzazione punteggio  
- Invio telematico  

# 5. Gestione Fiscale Periodica
- Versamenti F24  
- Acconti e saldi imposte  
- Ravvedimenti  
- Comunicazioni obbligatorie  
- Calendario scadenze personalizzato
