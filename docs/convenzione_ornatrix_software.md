# Servizi alle Imprese – Convenzioni CNA Roma - Ornatrix – Software Gestionale per Parrucchieri

Questa convenzione fa parte delle **Convenzioni CNA Roma per le Imprese**.

Per la panoramica completa: [Convenzioni](convenzioni.md)

## Ambito

Software gestionale dedicato ai saloni di parrucchieri ed estetisti.

## Destinatari

- Saloni di parrucchieri associati CNA Roma
- Centri estetici associati CNA Roma
- Barbieri e professionisti del settore bellezza

## Oggetto della Convenzione

Condizioni economiche dedicate e demo gratuita del software gestionale Ornatrix.

### Funzionalità del Software

**Gestione Prenotazioni**:
- Calendario appuntamenti digitale
- Promemoria automatici ai clienti
- Gestione sale e postazioni
- Prenotazioni online

**Anagrafica Clienti**:
- Database clienti completo
- Storico trattamenti
- Preferenze e note personali
- Sistema fedeltà e promozioni

**Fatturazione**:
- Emissione fatture elettroniche
- Gestione pagamenti
- Incassi e scontrini
- Collegamento con POS

**Organizzazione Attività**:
- Gestione magazzino prodotti
- Schede tecniche trattamenti
- Statistiche e report
- Gestione personale

## Vantaggi per gli Associati

- Condizioni economiche agevolate riservate a CNA
- Demo gratuita per testare il software
- Software specifico per il settore parrucchieri
- Supporto tecnico dedicato
- Aggiornamenti inclusi

## Come Attivare la Convenzione

### Requisiti

- Essere salone/centro associato CNA Roma
- Essere in regola con la quota associativa
- Possedere tessera associativa valida

### Attivazione

**Richiesta Demo Gratuita**:
1. Contattare CNA Roma indicando interesse per Ornatrix
2. CNA Roma fornisce i riferimenti del partner
3. Richiesta demo gratuita con condizioni CNA
4. Test del software senza impegno

**Attivazione Abbonamento**:
1. Dopo la demo, valutazione del servizio
2. Richiesta condizioni economiche agevolate CNA
3. Sottoscrizione abbonamento alle tariffe riservate
4. Formazione all'utilizzo del software
5. Assistenza e supporto continuativo

## Note

- La demo gratuita permette di testare tutte le funzionalità
- Le condizioni agevolate sono riservate agli associati CNA Roma
- Verificare con CNA Roma le tariffe aggiornate
- Il software è utilizzabile su PC, tablet e smartphone
- Include aggiornamenti normativi (fatturazione elettronica, GDPR, ecc.)
